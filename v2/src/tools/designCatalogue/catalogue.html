Design Catalogue

v2 design tokens — colour, typography, icon, space, radius, shadow, motion.

Semantic colours

Theme-aware tokens used by every component. Switch themes above to see them re-resolve.

Raw palette

Building-block tokens (--c-*). Themes map semantic vars onto these.

Typography

7 sizes × 4 weights. All sizes share the font-small slope, so ratios stay constant across viewport widths.

Icons

4 sizes in em units — icons scale with the surrounding text font-size.

Spacing

6 fixed-pixel spacing tokens used for gap / padding / margin.

Border radius

4 corner-radius tokens.

Elevation shadows

3 shadow tokens. Dark themes override these in their theme file for visibility on dark surfaces.

Phase 2 — Primitives

Flag

Country flag image. Height in em so it scales with surrounding text. Default code IL.

Icon

Inline SVG, size tokens --icon-{xs,sm,md,lg}. currentColor stroke so icons inherit their parent's colour.

Badge

Small inline marker. Medal variants (gold/silver/bronze) match the league-table medal colours.

Pill

Fully rounded inline tag. Status + league-type variants drive the league cards and table headers.

Button

Variants: primary / secondary / ghost / danger. Sizes sm/md/lg. btn--icon, btn--block, btn--pill modifiers.

Link

Variants: default (accent + underline), quiet (no underline until hover — table-link feel), strong, muted.

Avatar

Circular avatar — image or auto-initials. Optional status dot.

Tooltip

CSS-only on hover/focus. Four placements. Hover the buttons below to test.

Chip

Inline tag with optional remove control. Used by filter UIs.

FormField

Labelled input/textarea/select with hint + error. Replaces ad-hoc admin input styles.

Phase 3 — Components

PlayerCell

Table cell composing Flag + name link + optional title badges (BMAB outlined / Championship gradient).

StatusChip

Status pill with a glowing dot. Used on hero cards for Running / This year / Inactive.

TypePill

League-type label (Doubling / Regular / UBC). Thin wrapper over primitive Pill.

MedalRow + RankBadge

Top-3 finishers get a tinted row background and a circular medal badge.

ScoreCell

Numeric cell coloured by value position inside a [min, max] range. Switch themes to see the scale re-tint.

FilterPill

Toggleable chip — click to select / deselect.

ChartTooltip

Info panel that sits under a chart and shows details for the current data point.

Breadcrumbs

Segmented chevron-pill trail. Last crumb is the current page (accent-coloured).

SearchBox

Text input with popover results. Try typing a player name.

Navigation

Site nav bar with home, leagues dropdown, and player search. Click "Leagues ▾".

ColorScale

Red → amber → green gradient for data-emphasis cells. Re-tints per theme via dark-mode anchors.

LeagueHero

V13 (Lichess title bar) and V16 (wide hero banner).

PlayerHero

V7 (Lichess card) and V12 (hero banner with title ribbons + watermark).

PlayerBarChart

Canvas bar chart with moving-average overlay. Hover or click a bar to see details.

ThemePicker · AdminButton · ExportButton

Floating chrome. ThemePicker (bottom-right) and AdminButton (bottom-left) are mounted to the page body; the inline button below previews ExportButton.

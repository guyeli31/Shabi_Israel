League Page Header

Five layouts for the league page top-strip

Same five data points — league name, league type pill, start date, status pill, last-updated timestamp — rendered in five different idioms. Every variant uses the same fluid root font-size: clamp(1.05rem, 0.85rem + 0.6vw, 1.35rem) from design-lab.html, with every child in em units so the whole header scales linearly with viewport width. The type and status pills are the actual production components.

5 GitHub / Linear — every segment is a sibling switcher
GitHub · Linear · Notion

Monospace, slash-separated, productivity-app aesthetic. Each segment is both a link (click name = go up) AND a switcher (click chevron = pick sibling). Lean and very keyboard-friendly.

TradeoffReads as "developer tool" not "sports site" — feels off-brand for chess.
6 Transfermarkt — bordered bar with crest/flag icons
Transfermarkt · FIDE

Classic sports-stats site treatment: bordered bar, accent stripe down the left edge, and small badge/flag icons for each level. Feels authoritative and data-dense.

TradeoffOld-school aesthetic; can feel heavy on top of the existing site nav.
